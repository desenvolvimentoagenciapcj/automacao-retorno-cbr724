┌────────────────────────────────────────────────────────────────────────────┐
│ 🚀 COMO USAR (Simples e Direto!)                                          │
└────────────────────────────────────────────────────────────────────────────┘

   🟢 INICIAR (3 MODOS):
   
      • INICIAR_MONITOR.bat
        → Pergunta qual modo você quer (visível/minimizado/oculto)
      
      • INICIAR_MONITOR_MINIMIZADO.bat
        → Inicia direto minimizado na barra de tarefas
      
      • INICIAR_MONITOR_OCULTO.bat
        → Inicia totalmente oculto (RECOMENDADO para rodar 24/7)
   
   📊 STATUS:   Duplo clique em → STATUS_MONITOR.bat
                (verifica se está rodando)
   
   🔴 PARAR:    Duplo clique em → PARAR_MONITOR.bat
                (ou pressione Ctrl+C se estiver visível)
   
   📁 PROCESSAR: Coloque arquivos .ret em:
                D:\Teste_Cobrança_Acess\Retorno\
   
   ✨ PRONTO! Sistema processa automaticamente!

┌────────────────────────────────────────────────────────────────────────────┐
│ 📁 COMO FUNCIONA                                                           │
└────────────────────────────────────────────────────────────────────────────┘

   Você coloca o arquivo:
   📄 CBR7246250110202521616_id.ret
   ↓
   Em: D:\Teste_Cobrança_Acess\Retorno\
   
   O sistema AUTOMATICAMENTE:
   ✓ Detecta o arquivo
   ✓ Extrai a data do arquivo (ex: 01/10/2025)
   ✓ Processa todos os títulos
   ✓ Atualiza o banco Access
   ✓ Move para: Processados\CBR...processado_20251008_153045.ret
   
   Se houver erro:
   ✗ Move para: Erro\CBR...erro_20251008_153045.ret

┌────────────────────────────────────────────────────────────────────────────┐
│ 🎯 VERIFICAR SE ESTÁ FUNCIONANDO                                           │
└────────────────────────────────────────────────────────────────────────────┘

   ✅ Monitor está rodando?
      → Veja a janela preta com "👀 Aguardando novos arquivos..."
   
   ✅ Arquivo foi processado?
      → Procure em: Retorno\Processados\
   
   ✅ Houve erro?
      → Procure em: Retorno\Erro\
      → Abra: logs\monitor_retornos.log
   
   ✅ Dados no Access?
      → Abra dbBaixa2025.accdb
      → Veja tabela pcjTITULOS
      → Confira campo DT_PGTO_TIT (deve ter a data do arquivo!)

┌────────────────────────────────────────────────────────────────────────────┐
│ 💡 DICAS                                                                   │
└────────────────────────────────────────────────────────────────────────────┘

   • Feche o Access antes de processar (evita conflitos)
   
   • Arquivos IEDCBR são IGNORADOS automaticamente
   
   • O sistema cria BACKUP automático antes de processar
   
   • Os logs ficam em: logs\monitor_retornos.log
   
   • Para processar vários arquivos de uma vez:
     Coloque todos na pasta Retorno\ - o sistema processa um por um!

┌────────────────────────────────────────────────────────────────────────────┐
│ 🆘 PROBLEMAS?                                                              │
└────────────────────────────────────────────────────────────────────────────┘

   ❌ "Arquivo não é processado"
      → Veja logs\monitor_retornos.log
      → Confirme que é arquivo .ret
      → Confirme que monitor está rodando
   
   ❌ "Access está aberto"
      → Feche o Microsoft Access
      → Tente novamente
   
   ❌ "Erro de permissão"
      → Execute INICIAR_MONITOR.bat como Administrador
      (Clique com botão direito → Executar como Administrador)

   2. Teste manual primeiro: INICIAR_MONITOR.bat
   3. Veja mais detalhes em: LEIA-ME.txt ou RESULTADO_LIMPEZA.txt
